Main File Structure

All the file are well organized, its so easy to work with the template.
1. Unzip the files.
2. Open "beyond_hotel" folder.
You will find the two folder one is for lite version and other is for dark version file on the root that will be edited.
Also you will see 5 directories, css, js, fonts, img and scss.
In the first folder Content you will find style.css and other styling files the all the other files expect the html.
Js folder contain only the jquery library. Fonts folder contain essential font files for fonts.
